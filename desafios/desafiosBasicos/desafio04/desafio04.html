O Super Desafio: Totem de Aeroporto Pro

Neste desafio, o programa vai primeiro identificar quem está acessando o totem e depois perguntar o destino, tratando os textos para evitar erros.

  1. Identificação de Acesso: Peça para o usuário digitar o cargo (ex: "Piloto", "Comissário" ou "Passageiro").
  2. Tratamento de String: Use .toLowerCase() em todas as entradas para que o código aceite "PILOTO", "piloto" ou "Piloto".
  3. Switch de Nível de Acesso:
    • Se for "piloto" ou "comissário": Exiba "Acesso liberado à área de tripulação. Boa viagem!".
    • Se for "passageiro": O código deve abrir um segundo prompt perguntando o destino.
  4. Switch de Destino (apenas para passageiros):
    • "são paulo" ou "sao paulo": "Portão A - Embarque imediato".
    • "nova york": "Portão B - Aguarde a chamada".
    • "paris": "Portão C - Voo atrasado em 30 min".
    • Default: "Destino não listado no totem. Vá ao guichê físico".
  5. Default Geral: Se o cargo não for nenhum dos três, exiba "Acesso negado. Procure a segurança".
