O Desafio: Simulador de Crédito

Crie um novo arquivo HTML ou use um bloco script para desenvolver a seguinte lógica:

  • Entrada de Dados: Peça o nome do cliente, a idade e o valor do salário mensal.
  • Regras de Aprovação: Para o empréstimo ser aprovado, o cliente precisa atender a dois critérios simultâneos (use operadores lógicos):
    1. Ter idade entre 21 e 60 anos.
    2. Ganhar um salário maior ou igual a R$ 2.000,00.
  • Saída de Dados:
    • Use window.alert para dar as boas-vindas ao cliente.
    • Use document.write para exibir o resultado final:
    • Se aprovado: "Parabéns [Nome], seu crédito foi aprovado!"
    • Se reprovado: "Desculpe [Nome], você não atende aos requisitos mínimos."
  • Bônus (Opcional): Use o typeof para exibir no console ou na página o tipo das variáveis que você capturou.
